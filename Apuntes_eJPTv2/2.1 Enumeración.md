# 🛡️ Enumeración de Servicios
La fase de enumeración es donde buscamos vectores de ataque específicos. El objetivo es identificar versiones de software, nombres de usuario, carpetas compartidas o configuraciones por defecto.

## 🚀 Metodología con [[Metasploit#🛠️ Enumeración de Servicios|Metasploit]]
Para todos estos servicios, utilizaremos módulos **auxiliary/scanner/**. La ventaja de usar Metasploit es que los resultados se pueden almacenar en la base de datos interna para su uso posterior.

| **Servicio** | **Puerto** | **Qué buscamos**                                | **Enlace a Módulos**                                              |
| ------------ | ---------- | ----------------------------------------------- | ----------------------------------------------------------------- |
| **FTP**      | 21         | Acceso anónimo, versiones con backdoors.        | [[Metasploit#📂 FTP (File Transfer Protocol)]]                    |
| **SSH**      | 22         | Versión de banner, enumeración de usuarios.     | [[Metasploit#🔑 SSH (Secure Shell)]]                              |
| **HTTP**     | 80/443     | Directorios, métodos permitidos, versiones.     | [[Metasploit#🌐 Web Server - HTTP (HyperText Transfer Protocol)]] |
| **SMB**      | 445        | SMBv1, recursos compartidos, nombre de dominio. | [[Metasploit#📁 SMB (Server Message Block)]]                      |
| **MySQL**    | 3306       | Root sin pass, versiones, tablas expuestas.     | [[Metasploit#📊 MySQL (Structure Query Language)]]                |
| **SMTP**     | 25         | Enumeración de usuarios internos (VRFY).        | [[Metasploit#📧 SMTP (Simple Mail Transfer Protocol)]]            |
