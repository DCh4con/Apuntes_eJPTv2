# 🚀 Sugerencia de Exploits (Escalada de Privilegios)

La escalada de privilegios consiste en aprovechar una vulnerabilidad, un error de configuración o una debilidad en el sistema operativo para elevar los permisos del usuario actual.

## 🛠️ Metodología con [[Metasploit]]
Metasploit ofrece una herramienta automatizada muy potente que analiza el sistema operativo de la víctima y lo compara con su base de datos de exploits locales.

- Módulo: `post/multi/recon/local_exploit_suggester`

Este módulo es multiplataforma (**Windows y Linux**) y es fundamental para identificar vectores de ataque rápidos.

**Requisitos:**

- Tener una sesión activa en Metasploit (preferiblemente de [[Meterpreter]]).
- La sesión debe ser de un usuario con privilegios bajos.

**Procedimiento:**

1. **Identificar la sesión:** Mira tus sesiones abiertas con el comando `sessions`.
2. **Cargar el módulo:** `use post/multi/recon/local_exploit_suggester`
3. **Configurar:** `set SESSION <id_sesion>`
4. **Ejecutar:** `run` o `exploit`

> [!important] **Nota Clave:** Este módulo **no ejecuta los exploits**, solo los sugiere. Te devolverá una lista de módulos que tienen una alta probabilidad de éxito basándose en los parches de seguridad (o la falta de ellos) del sistema.
